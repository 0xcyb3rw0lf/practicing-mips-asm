List of Programs to practice:

[X] Read/Print String  
[] Read/Print Integers  
[] Bubble Sort  
[] Add and Subract Two Integers  
[] Divide Two Integers, Get Remainder and Quotient  
[X] formula-1 (basic addition and subtraction + load and store from memory)
